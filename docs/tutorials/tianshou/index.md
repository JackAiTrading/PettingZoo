## 天授（Tianshou）概述

[天授（Tianshou）](https://github.com/thu-ml/tianshou)是一个轻量级的强化学习平台，提供快速、模块化的框架和 Python 风格的 API，能够用最少的代码行数构建深度强化学习智能体。
它完全基于 [PyTorch](https://pytorch.org/) 构建，仅用约 4000 行代码实现。

它拥有大量的算法实现和高质量的软件工程标准：全面的测试、类型提示和完整的文档。
